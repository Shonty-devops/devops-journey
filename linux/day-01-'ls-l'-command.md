## What is `clear`?

`clear` is one of the basic Linux/Unix commands used to clear 
the contents present on a screen. It is used across all Linux 
and Unix systems and it helps in removing all the previously used commands
and use a terminal as a clean slate.

## What is `ls -l`?

`ls -l` is one of the basic Linux/Unix commands used to get 
 us more information or content.
 
 '-' is used when we want to let the system know that we 
 are going to use an argument or pass an argument. Meanwhile
 using '--' can be used to pass an argument with more than
 one letter. 

---

## Commands practised

| Command | Output behaviour |
|---|---|
| `clear` | Clears all the commands or activities done on terminal |
| `ls -l` | Lists the details of the file |
